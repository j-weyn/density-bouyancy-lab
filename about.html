About This Learning Hub

This interactive platform was created to help students understand fluid mechanics through hands-on simulations, guided practice, and quiz assessments.

Students can explore density, buoyancy, pressure, and floating behavior in a visual environment before testing their knowledge through the quiz mode.

Once a student finishes the quiz, their result is recorded in the admin dashboard so instructors can review progress and performance.

What You Can Do

  • Run the density and buoyancy simulation
  • Take the quiz as a student and submit answers
  • View results from the admin dashboard
  • Learn through guided concepts and interactive examples
